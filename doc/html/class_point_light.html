Detailed Description

Represents a point light source in a 3D graphics environment.

Inherits from Light class. This class encapsulates the properties and functionalities of a point light source, which emits light in all directions from a single point. It includes properties for ambient, diffuse, and specular lighting as well as a cutoff value for the light's influence.
