Detailed Description

Represents a fake shadow for a 3D object.

Inherits from Mesh class. This class is used to create and manage fake shadows in a 3D environment. It allows the creation of shadows that are not physically accurate but visually effective for certain rendering scenarios. It links with a specific model and can be manipulated in terms of offset and scaling.
