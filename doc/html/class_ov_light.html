Detailed Description

Represents a light object with different subtypes.

This class is a simplified version of OvLight, focusing solely on the subtypes of lights that can be represented. It includes an enumeration for different kinds of light such as Omni, Directional, and Spot lights.

Enumerates the different subtypes of lights.
