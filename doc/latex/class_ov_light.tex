\doxysubsection{Detailed Description}
Represents a light object with different subtypes. 

This class is a simplified version of \doxylink{class_ov_light}{Ov\+Light}, focusing solely on the subtypes of lights that can be represented. It includes an enumeration for different kinds of light such as Omni, Directional, and Spot lights.
