\doxysubsection{Detailed Description}
Represents a light source in a 3D scene. 

Inherits from the \doxylink{class_node}{Node} class. This class encapsulates properties and behaviors of a light source, including ambient, diffuse, and specular lighting components, as well as attenuation factors. It also supports setting and getting the light\textquotesingle{}s position in the scene.
