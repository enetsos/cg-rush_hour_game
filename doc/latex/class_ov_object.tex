\doxysubsection{Detailed Description}
Represents an object with various types in a graphics or game engine context. 

This class is a simplified version of \doxylink{class_ov_object}{Ov\+Object}, focusing on enumerating different types of entities that can be represented. It includes an enumeration for foundational types, derived classes, animation, physics-\/related entities, and a terminator.
