\doxysubsection{Detailed Description}
Represents a 3D mesh in a graphics environment. 

Inherits from \doxylink{class_node}{Node}. This class encapsulates a 3D mesh, which is a collection of vertices, and a material. It provides functionalities to add vertices, render the mesh, and manage its material and shadow casting properties.
