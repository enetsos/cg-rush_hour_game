\doxysubsection{Detailed Description}
Manages a collection of \doxylink{class_node}{Node} objects for rendering and interaction. 

Inherits from the \doxylink{class_object}{Object} class. This class is designed to handle a list of \doxylink{class_node}{Node} objects, providing functionalities to add, render, and manage these nodes in a 3D graphics environment.
