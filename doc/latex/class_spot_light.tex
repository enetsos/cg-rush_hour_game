\doxysubsection{Detailed Description}
Represents a spot light source in a 3D graphics environment. 

Inherits from \doxylink{class_point_light}{Point\+Light} class. This class encapsulates the properties and functionalities of a spot light source, which emits light in a specific direction with a conical spread. It includes properties for ambient, diffuse, and specular lighting as well as a cutoff value for the light\textquotesingle{}s influence and the direction of the light.
