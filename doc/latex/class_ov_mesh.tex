\doxysubsection{Detailed Description}
Represents a mesh object with different subtypes. 

This class is a simplified version of \doxylink{class_ov_mesh}{Ov\+Mesh}, focusing on the different subtypes of meshes that can be represented. It includes an enumeration for different kinds of meshes, such as Default, Normal\+Mapped, and Tessellated.
