\doxysubsection{Detailed Description}
Represents a texture in a 3D graphics environment. 

Inherits from \doxylink{class_object}{Object}. This class encapsulates the properties and functionalities related to a texture, including its ID, anisotropic filtering capabilities, and texture file handling. It is used to manage and apply textures to objects in the scene.
