\doxysubsection{Detailed Description}
Represents a directional light in a 3D scene. 

Inherits from \doxylink{class_light}{Light} class. Directional lights simulate light sources that are far away (like the sun) and emit light rays that are parallel and in a single direction. This class provides functionalities to set up and render a directional light in a scene.
