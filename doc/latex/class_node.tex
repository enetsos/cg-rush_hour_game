\doxysubsection{Detailed Description}
Represents a node in a scene graph for a 3D graphics environment. 

Inherits from \doxylink{class_object}{Object}. This class is a fundamental part of the scene graph, representing a node that can have child nodes and a parent node. It handles operations such as rendering, hierarchy management, transformation, and spatial queries.
