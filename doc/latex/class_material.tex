\doxysubsection{Detailed Description}
Represents a material in a 3D graphics environment. 

Inherits from the \doxylink{class_object}{Object} class. This class encapsulates material properties such as emission, ambient, diffuse, specular components, and shininess, which are used to determine how the material interacts with light. It also supports texture mapping.
