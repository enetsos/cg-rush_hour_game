\doxysubsection{Detailed Description}
The main engine class responsible for initializing and managing the 3D graphics environment. 

This class encapsulates functionalities for initializing the rendering environment, handling input, managing scenes, and controlling the render loop. It acts as a central hub for the graphics application.
